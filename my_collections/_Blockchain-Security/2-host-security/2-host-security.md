 此安全需求主要著重於主機與資料。首要重於保護主機與設備安全之控制措施。之後則考慮到資料。在私有區塊鏈應用當中，最重要的資料莫過於錢包 (與相關金鑰) 資料，以及區塊鏈資料。因此分別要求對此兩類資料進行盤點與保護。因此，這些設備也會需要透過相關的實體安全控制措施來進行保護。
